[Jekyll] arithmetic in conditional expression doesnt work

When I write in Jekyll, I want something like every second element using one piece of code and the others using other piece of code. So I wrote something like: 
      
    {% if loopindex % 2 == 0 %}
      // do something
    {% else %}
      // do something
    {% endif %}
But it doesn't work. After I search online, I realised that
  • First thing first, this is not liquid syntax. Liquid uses maths filter for arithmetic instead of maths symbol.
  • Filter can't be used in liquid if expression as pipes are not allowed in such a control structure. And this will finally resolve to {% if loopindex %} which is always true. To solve this problem, I have to assign the calculation to a variable then use it in if expression.
So the code should be like this. 
      
    {% assign i = loopindex | modulo: 2 %}
    {% if i == 0 %}
      // do something
    {% else %}
      // do something
    {% endif %}
