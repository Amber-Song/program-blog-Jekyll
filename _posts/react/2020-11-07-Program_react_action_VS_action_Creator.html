[React] Action VS Action creator [Question mark]

I was confused by the action and action creator. So in this article, I will explain how I understand it now.

In short, action is an object while action creator is a function. 
      
    // Action creator return an action
    const addWord = (word, meaning) => {
        const lastCheckTime = new Date().toGMTString();

        return {
            type: "ADDWORD",
            payload: {
            word,
            meaning,
            lastCheckTime,
            checkPeriod: 0
            }
        };
    };
addWord() is action creator. It is a function. And it creates action by returning an object. While that object is an action. 
        
    return <button onClick={() => dispatch(addWord(word, meaning))}>+</button>;
When the onClick function is triggered, it dispatches the action, or to say the object.

But there is one thing that I am still not that sure. When an action is dispatched, how could the react know which reducer is that if there are multiple reducers?
